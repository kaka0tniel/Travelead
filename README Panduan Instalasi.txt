Petunjuk untuk menjalankan aplikasi 

1.pada database buat database dengan nama jayatiket
2. pada terminal lakukan sintax php artisan migrate agar tbl pada aplikasi database tercipta
3.buka database, dengan cara manual pada tbl dtabase, buka tbl roles,baris pertama isi id=1,RuleName=Admin dan baris kedua isi id=2, RuleName=User
4.pada terminal jalankan php artisan serve
5. buka browser-> isi alamt url localhost:8000 untuk membuka halaman home aplikasi
6. untuk menjalankan aplikasi registrasi dulu sebagai user.
7. untuk mendaftar sebagai admin, daftar akun seperti biasa. setelah terdaftar buka databse->tbl users dan ubah roles_id menjadi 1 agar user tersebut menjadi admin
8. untuk mendaftar sebagai custummer/ user biasa, daftar akun dan tidak usah ubah-ubah lagi di databse karna otomatis terdaftar sebagai user biasa
9. Selamat Mencoba Aplikasi
